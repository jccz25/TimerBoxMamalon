# TimerBoxMamalon 🥊⏱️

Timer de intervalos para box, MMA y entrenamientos funcionales. Hecho en Celaya, Gto 🇲🇽

## Features
- Configura Sets, Ejercicios, Tiempo de trabajo y descanso
- Descanso entre sets configurable
- Botones PLAY / PAUSE / STOP
- 100% offline, sin anuncios, sin rastreadores

## Descargar
Puedes compilar el proyecto en Android Studio o descarga el APK desde [Releases](https://github.com/jccz25/TimerBoxMamalon/releases).
